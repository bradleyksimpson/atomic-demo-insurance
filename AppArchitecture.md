# Demo Insurance App Architecture

## Key Features Based on Industry Leaders

### Core Features
1. **AI-Powered Chatbot** (Maya-style from Lemonade)
   - Conversational interface for quotes
   - Claims processing assistance
   - Policy questions

2. **Telematics Dashboard** (Root/Progressive style)
   - Safe driving score
   - Trip tracking
   - Potential discounts display

3. **Photo Claims** (Allstate QuickFoto style)
   - Take photos of damage
   - AI damage assessment
   - Instant repair estimates

4. **Digital ID Cards**
   - Apple Wallet integration
   - Quick access from home screen

5. **Smart Home Integration**
   - IoT device monitoring
   - Preventive alerts
   - Risk mitigation tips

### Insurance Categories
- House Insurance
- Contents Insurance
- Vehicle Insurance
- Pet Insurance
- Life Insurance
- Travel Insurance

### Design Philosophy
- Minimalist UI (Lemonade-inspired)
- Pink accent color (#FF0083)
- Conversational UX
- Mobile-first approach
- One-tap actions where possible

### Technical Stack
- SwiftUI for UI
- Combine for reactive programming
- Core Motion for telematics
- Vision framework for photo analysis
- MapKit for location services
