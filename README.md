# Demo Insurance iOS App

A comprehensive iOS demo application showcasing the most innovative features from leading insurance companies like Lemonade, Root, Oscar Health, and Allstate.

## 🚀 Features

### Core Innovation Features
- **AI-Powered Assistant (Maya)**: Conversational interface for quotes and claims inspired by Lemonade
- **Telematics Dashboard**: Safe driving tracking and scoring system (Root/Progressive style)
- **Photo Claims Processing**: Instant damage assessment with AI (Allstate QuickFoto style)
- **Digital ID Cards**: Apple Wallet integration for instant access
- **Smart Home Integration**: IoT device monitoring for risk reduction

### Insurance Categories
- 🏠 **House Insurance**: Property protection with smart home discounts
- 📦 **Contents Insurance**: Personal belongings coverage
- 🚗 **Vehicle Insurance**: Auto coverage with telematics
- 🐕 **Pet Insurance**: Comprehensive pet health coverage
- ❤️ **Life Insurance**: Family protection plans
- ✈️ **Travel Insurance**: Trip protection and assistance

### Advanced Features
- **Predictive Analytics**: Risk assessment and prevention
- **Real-time Monitoring**: Connected device alerts
- **Gamified Experience**: Safe driving rewards
- **Instant Quotes**: 90-second quote generation
- **Fraud Detection**: AI-powered claim validation

## 🎨 Design Philosophy

- **Minimalist UI**: Clean, white backgrounds with strategic color usage
- **Lemonade-Inspired**: Pink accent color (#FF0083) for CTAs and highlights
- **Mobile-First**: Optimized for iPhone with intuitive gestures
- **Conversational UX**: Natural language interactions
- **One-Tap Actions**: Streamlined user flows

## 📱 App Structure

```
DemoInsurance/
├── Models/
│   └── Models.swift          # Data models for policies, claims, etc.
├── Views/
│   ├── ContentView.swift     # Main tab navigation
│   ├── OnboardingView.swift  # Welcome flow
│   ├── HomeView.swift        # Dashboard with categories
│   ├── ChatbotView.swift     # AI assistant (Maya)
│   ├── TelematicsView.swift  # Driving insights
│   ├── ClaimsView.swift      # Photo claims processing
│   ├── PoliciesView.swift    # Policy management
│   ├── ProfileView.swift     # User profile & settings
│   └── SmartHomeView.swift   # IoT device management
├── Resources/
│   └── Theme.swift           # Color scheme & styling
└── Info.plist               # App permissions & config
```

## 🔧 Technical Requirements

- iOS 16.0+
- Xcode 15.0+
- SwiftUI
- Core Motion (for telematics)
- Vision Framework (for photo analysis)
- PhotosUI (for image selection)
- Charts Framework (for data visualization)

## 📋 Permissions Required

- **Camera**: For claims photo capture
- **Photo Library**: For uploading damage photos
- **Location (When In Use)**: For telematics and location-based services
- **Motion**: For driving behavior tracking

## 🏗️ Getting Started

1. Open `DemoInsurance.xcodeproj` in Xcode
2. Build and run on iOS Simulator or device
3. Grant required permissions when prompted
4. Experience the innovative insurance features!
