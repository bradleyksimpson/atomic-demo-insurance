## 📱 Running the App

### Option 1: Using Xcode (Recommended)
1. Open `DemoInsurance.xcodeproj` in Xcode
2. Select any iPhone simulator from the device menu (iPhone 15 Pro recommended)
3. Press **Cmd+R** or click the ▶️ Run button
4. The app will launch in the iOS Simulator

### Option 2: Command Line Launch
```bash
cd "/Users/bradleysimpson/Demo Insurance"
xcodebuild -project DemoInsurance.xcodeproj -scheme DemoInsurance -destination 'platform=iOS Simulator,id=A9999E94-0FD9-41FA-B562-1CD294C68891' run
```

## 🔧 Troubleshooting

If you encounter any issues:
1. **Simulator not found**: Choose any available iPhone simulator
2. **Build errors**: Clean build folder (Cmd+Shift+K) and rebuild
3. **Missing simulator**: Install additional simulators in Xcode → Window → Devices and Simulators
